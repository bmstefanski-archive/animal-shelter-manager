## How to use it? ##   
You can find a few commands below, if you found any **typo** please inform me.  

| Command                             | Description                         |
| ----------------------------------- | ----------------------------------- |
| !add                                | Add your pet                        |
| !remove                             | Remove your pet                     |
| !status                             | Check shelter's status              |
| !list                               | Display all pets with their **ID**s |
| !animal                             | Statistics                          |
| !save                               | Save and exit                       |
